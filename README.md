# IgnisCura OS 🛡️🔥
**Next-Gen Tactical Incident Command Dashboard**

IgnisCura OS is a high-performance tactical interface designed for **Incident Commanders (IC)** to monitor real-time biometric and environmental telemetry of active firefighter crews. Built with a scalable architecture, it features tactical multi-crew switching, live health indicators (Heart Rate, Core Temp), and instant mass evacuation controls.

## 🚀 Key Tactical Features
*   **Two-Tier Command Structure:** Built to align with NIMS (National Incident Management System) standards.
